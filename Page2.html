Веб-приложение CRUD
Описание:
Выполнение операций CRUD (Создание-чтение-Обновление-удаление) является очень распространенной функцией для большинства веб-сайтов (таких как блоги, электронная коммерция, информационные панели и так далее).
Для разработки приложения CRUD использовались языки программирования как JavaScript, PHP, MySQL. В приложении реализовано создание, удаление, изменение статей на каждой странице. Регистрация модераторов (субадминов) реализована с преоритетом, только главный администратор может регистрировать пользоватлей. А также на сайте реализована обратная связь пользователя с администратором, для коммереских предложений.

Для данного сайта зарнее был создан референс по которому создавалось приложение.
