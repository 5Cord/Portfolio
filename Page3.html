Веб-сайт, подключенный к API
Описание:
API (Application Programming Interface) – это набор правил, по которым одно приложение может взаимодействовать с другим. В контексте веб-разработки API позволяет вам получать данные от сторонних сервисов и использовать их на вашем сайте или отправлять данные на сторонние сервисы.
Разрабатывая данный проект, разработчик ипользовал открытый, бесплатный API (restcountries.com). Приложение построенно на взаимодействие пользователя и определнным набором правил, который прописал разработчик. Данное API даёт доступ к большой базе странн и её полной информации (континет, столица, страна, язык, валюта).
